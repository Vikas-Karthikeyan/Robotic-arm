# Robotic-arm

In modern industries, automation is essential for improving efficiency, precision, and 
productivity. Traditional sorting systems either require human intervention or 
mechanical sorting techniques that lack flexibility and accuracy. Manual sorting is 
prone to human errors and inefficiencies, while mechanical methods often struggle 
with varying object characteristics. 
Colour-based sorting is widely used in industries such as packaging, food processing, 
and recycling. A smart, automated sorting system can significantly enhance 
throughput, reduce errors, and optimize the overall process. The combination of 
computer vision and robotic automation provides an advanced solution to 
conventional sorting challenges. 
This project aims to develop an intelligent colour-based sorting system using: 
• ESP32-CAM for real-time colour detection. 
• OpenCV for image processing and classification of objects based on color. 
• An IR sensor for object detection and activation of the sorting process. 
• A 4-DOF robotic arm for automated sorting and placement of objects into 
designated bins. 
By leveraging these technologies, the system can accurately classify and sort objects 
based on predefined colour categories, improving efficiency in industrial automation.
